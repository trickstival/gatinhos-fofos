HTML:


span

A tag span é uma tag genérica para elementos inline, ou seja,
não surte efeito direto no comportamento do elemento.

Exemplo:

Eu sou um span

table

A tag table tem a função de criar tabelas no documento HTML.

Ela pode conter elementos filhos que ajudam em sua formatação, sendo estes:

  • thead
  • tbody
  • tr
  • th
  • td

O thead define um cabeçalho, ou um título para a tabela. O tbody indica o início do corpo da tabela, funcionando como
um elemento em bloco que o agrupa. O tr define uma linha da tabela, podendo estar dentro dele tanto o td, quanto o th.
O th indica uma coluna agrupada ao thead para (o th é posto dentro do thead), formando uma célula no cabeçalho. O td representa a junção linha-coluna agrupada ao tr (o td é posto dentro do tr).

Exemplo:

Eu sou um th
Eu sou um td Eu sou outro td

nav

A tag nav foi adicionada no HTML 5 para representar uma section de navegação. Um exemplo de nav é o próprio menu chamado Navegação deste site.

Exemplo:

Esta tag não apresena nenhum conteúdo em especial, mas surte efeito quando se trata de SEO.
